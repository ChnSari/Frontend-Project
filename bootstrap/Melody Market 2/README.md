# Melody Market 2

Modern music instrument sales interface — Responsive, minimal, and user-focused UI.

---

## Project Overview

**Melody Market 2** is a modern and stylish music instrument sales web interface.

This project was developed to:

* Practice modern UI design
* Develop responsive layouts
* Use the Bootstrap grid system
* Create a frontend portfolio project

---

## Features

✅ Modern and minimal design  
✅ Fullscreen hero section  
✅ Responsive navbar  
✅ Bootstrap 5 grid system  
✅ Product cards with hover effects  
✅ About section  
✅ Contact form  
✅ Footer section  
✅ Google Fonts integration  
✅ Clean user experience  

---

## UI Components

### Navbar

* Fixed top navigation
* Responsive hamburger menu
* Minimal logo area

### Hero Section

* Fullscreen background
