# 🔗 LinkedIn Clone UI — Bootstrap 5

<p align="center">

A modern and responsive **LinkedIn UI clone**.
This project recreates the core LinkedIn layout using Bootstrap 5 and modern web technologies.

</p>

---

##  About The Project

This project aims to recreate the core user interface of the LinkedIn platform using modern frontend technologies.

###  Purpose

* Practice modern UI development
* Learn Bootstrap grid system usage
* Build component-based layouts
* Improve responsive frontend development skills
