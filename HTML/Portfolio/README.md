#  Cihan Sarı – Personal Portfolio


> A minimal, paper-inspired personal portfolio with a clean and professional aesthetic.

---

##  About The Project

This project is designed as a personal portfolio with a **paper / notebook-inspired aesthetic**,  
offering a warm, clean, and typography-focused experience.

Unlike highly animated or overly complex portfolio designs, this project focuses on:

- Simplicity
- Readability
- Professional presentation
- Calm and minimal UI
- Clear content hierarchy

---

##  Features

- Paper-style layout design
- Serif + Sans-serif typography balance
- Skill representation based on level (minimal approach)
- Clean project presentation
- Fully responsive structure
- Distraction-free user interface
