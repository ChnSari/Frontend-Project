


#  PLAYFLIX — Movie Page UI



A modern **Netflix-inspired movie page design** built using pure HTML and CSS.

This project demonstrates a cinematic user interface with a hero banner, movie cards, and hover interactions — designed for streaming platforms, movie catalogs, or UI practice.

---

##  Features

*  Netflix-style UI layout
*  Dark mode design
*  Hero banner with gradient overlay
*  Movie card grid system
*  Image hover zoom effect (no shadow)
*  Modern typography (Poppins)
*  Responsive layout structure
*  Pure HTML + CSS (no frameworks)

---

##  Design Concept

The interface is designed to provide a cinematic experience:

* Streaming platform look
* Minimal dark theme
* Large visual hero section
* Interactive movie cards
* Clean and modern layout

---

##  Technologies Used

* HTML5
* CSS3
* Google Fonts (Poppins)
* CSS Grid
* CSS Variables
* Hover animations
