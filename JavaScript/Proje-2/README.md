#  TaskFlow Pro

A modern, fast, and user-friendly task management application.
Organize your daily tasks بسهولة with a clean design and powerful features.

---

##  Features

*  Add / delete tasks
*  Toggle task completion
*  Real-time search
*  Task filtering:

  * All
  * Active
  * Completed
*  Data persistence with LocalStorage
*  Bootstrap Toast notifications
*  Modern and responsive UI

---

##  Technologies Used

* HTML5
* CSS3 (Custom Properties + Flexbox + Grid)
* JavaScript (Vanilla JS)
* Bootstrap 5

##  Installation

To run the project locally:

```bash
git clone https://github.com/kullaniciadi/taskflow-pro.git
```

Then simply open:

 `index.html` in your browser

---

##  How It Works

* All tasks are stored in `localStorage`
* Data persists even after refreshing the page
* Filtering and search operations work instantly
