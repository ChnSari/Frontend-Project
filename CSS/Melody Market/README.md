

#  Melody Market

![Melody Market Preview](img/1.png)

A modern and stylish music instrument e-commerce website.

Melody Market is a modern web interface developed to provide music enthusiasts with high-quality instruments. It offers a responsive design, clean UI, and a user-focused experience.

---

##  Features

✅ Modern and minimal design
✅ Fully responsive layout (mobile-friendly)
✅ Hero section with product card system
✅ CSS Flexbox layout implementation
✅ Google Fonts integration
✅ Clean and readable code structure
✅ Footer contact form

---

##  Technologies Used

* HTML5
* CSS3
* Google Fonts
* Flexbox Layout
